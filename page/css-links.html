Links

Links can be styled with any CSS property (e.g. color, font-family, background, etc.).
In addition, links can be styled differently depending on what state they are in.

When setting the style for several link states, there are some order rules:

  • a:hover MUST come after a:link and a:visited
  • a:active MUST come after a:hover

The four links states are:

  • a:link
  • a:visited
  • a:hover
  • a:active
a:link

A normal\unvisited link, The style will be apply if the user haven't clicked or hover the mouse over it.

Example:
Click Here!
Note: We have changed the link color to green, If the link isn't green that's mean that you already clicked it.
a:visited

A link the user has visited, The style will be apply if the user clicked on the link.

Example:
Click Here!
Note: We have changed the visited color to light blue.
a:hover

The style will be apply if the user hover the mouse over the link.

Example:
Hover Here!
Note: We have changed the hover color to light green.
a:active

The style will be apply if the user holding the link.

Example:
Click And Hold Here!
Note: We have changed the active color to dark blue.

What is the correct way to change the hover color of all the links to red?
