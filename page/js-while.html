While Loop

The while loop loops through a block of code as long as a specified condition is true.

Example:

    while (i < 5) {
        text += "The number is " + i + "<br>";
        i++;
    }
Result:

The number is 0
The number is 1
The number is 2
The number is 3
The number is 4

In the example above, the code in the loop will run, over and over again, as long as variable 'i' is less than 5.
