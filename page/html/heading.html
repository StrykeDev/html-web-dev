HTML H1-6 Tags

Use h1 to h6 elements only for headings.
Do not use them just to make text bold or big, there are other tags for that.

This is <h1> heading.

This is <h2> heading.

This is <h3> heading.

This is <h4> heading.

This is <h5> heading.
This is <h6> heading.
